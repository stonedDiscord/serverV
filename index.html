update your old

  1. Place serverV in the same folder as your old server
  2. Start serverV
  3. Press START button
  4. Check the Public Server Mode box to make your server appear on the serverlist.

install from scratch

  1. Copy AOs server folder whereever you want your server
  2. Inside that folder drop your copy of serverV
  3. Edit the settings files
Congratulations, your new serverV is now configured! Try this if you're having trouble

commands

/pm Godot sup send a pm
/ps oocpassword logs you in as a mod
/roll 6 rolls a dice

for mods

/ip show the playerlist with IPs
/kick Godot kick a player
/ban zettaslow ban a player
/mute Hawk mute a player
/unmute Hawk unmute a player
/ignore Doge ignore a player serverside
/unignore Doge continure processing his commands
/undj Vinyl keep this player from changing the music
/dj Vinyl allow it again
/play Pursuit(HY).mp3 play a track thats not on the list
/send Godot MU#-1#% send this command to the player

for admins

/reload reload the banlists,char files,etc.
/sendall BN#zetta#% send this command to all players
/lock 2 keep other mods out of your current area
/public 0 hide/show the server on the masterlist
/unban 12.34.56.78 unban this IP/HD
Warning: Mods can only be kicked by admins
